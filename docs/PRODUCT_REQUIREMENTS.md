# 소행성 실서비스 요구사항 v1

## 기본 정보
- 앱 이름: 소행성
- 안드로이드 패키지명: com.jinyoung.sohangseong
- 로그인: 카카오, 구글, 이메일

## MVP 범위
- 커뮤니티
  - 게시글 작성/수정/삭제
  - 댓글/대댓글
  - 반응(공감, 도움돼요, 응원)
  - 신고/차단
- 스탠다드
  - 진행중 투표 목록
  - 투표 참여
  - 인정템 목록 및 상세
- 프로필
  - 내 활동 요약
  - 차단회원 관리
  - 알림 설정
- 인증/계정
  - 소셜 로그인(카카오/구글)
  - 이메일 로그인/가입
  - 토큰 재발급

## 운영 필수
- 관리자 기능
  - 신고 처리
  - 콘텐츠 숨김/복구
  - 사용자 제재
- 모니터링
  - API 에러 추적
  - 앱 크래시 추적
  - 핵심 지표(가입, DAU, 게시글 수)

## 비기능 요구사항
- 응답시간: 주요 조회 API p95 500ms 이하
- 가용성: 월 99.5% 이상
- 보안
  - 비밀번호 해시 저장
  - 토큰 탈취 대응(리프레시 토큰 회전)
  - 입력값 검증 및 SQL 인젝션 방지

## 제외 범위(후속)
- 인앱 결제
- 추천 알고리즘 고도화
- 웹 관리자 고급 대시보드
